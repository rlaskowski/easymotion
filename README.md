# EasyMotion

## Run
```sh
$ make run
```

## Build
```sh
# Creating bin file in dist directory
$ make build
```

## Commands under easymotion bin file
``` sh
# Install as a system service
$ easymotion install
```
``` sh
# Uninstall 
$ easymotion uninstall
```
``` sh
# Start service 
$ easymotion start
```
``` sh
# Stop service 
$ easymotion stop
```
``` sh
# Restart service 
$ easymotion restart
```
